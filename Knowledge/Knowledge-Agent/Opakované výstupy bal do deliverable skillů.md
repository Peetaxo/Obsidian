# Opakované výstupy bal do deliverable skillů

## Trigger
- opakovaný PDF guide
- vizuální explainer
- marketingový carousel
- HTML dashboard nebo report

## Core rule
- Pokud stejný typ artefaktu vzniká opakovaně → vytvoř deliverable skill

## Decision rules
- Pokud výstup má stabilní formát → definuj skill pro vstup, strukturu a soubory
- Pokud výstup potřebuje brand → napoj ho na style skill
- Pokud výstup publikuje ven → odděl tvorbu od schválení
- Pokud výstup používá API → otestuj konfiguraci před automatizací
- Deliverable skill zvyšuje konzistenci výstupu
- Pevná struktura snižuje opakované ladění promptu

## Use for
- PDF lead magnety
- vizuální explainery
- social carousely
- interní HTML reporty

## Avoid
- dlouhý prompt pro každý opakovaný artefakt
- publikování bez schválení
- deliverable skill bez jasného výstupního formátu
