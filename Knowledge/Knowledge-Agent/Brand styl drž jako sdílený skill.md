# Brand styl drž jako sdílený skill

## Trigger
- opakované vizuální výstupy
- HTML dashboardy nebo reporty
- projekt má brand manuál
- výstupy driftují ve stylu

## Core rule
- Pokud agent opakovaně generuje vizuální výstupy → dej brand pravidla do sdíleného style skillu

## Decision rules
- Pokud výstupy mají stejný brand → použij jeden style skill
- Pokud výstupy vypadají nekonzistentně → odděl styl od konkrétního úkolu
- Pokud existuje design systém → odkaž ho ze skillu
- Sdílený styl snižuje opakované promptování
- Jedno místo pro barvy a typografii zvyšuje konzistenci
- Style skill nenahrazuje kontrolu konkrétního výstupu

## Use for
- HTML dashboardy
- PDF guide
- vizuální explainery
- marketingové assety

## Avoid
- kopírovat brand prompt do každého úkolu
- držet více nekompatibilních style skillů
- generovat brandované výstupy bez kontroly výsledku
