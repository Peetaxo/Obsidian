# Skill katalog audituj pravidelně

## Trigger
- mnoho nainstalovaných skillů
- duplicity nebo překryvy skillů
- agent neví který skill použít
- skill knihovna se pravidelně mění

## Core rule
- Pokud počet skillů ztěžuje orientaci → vytvoř katalog a audit skillů

## Decision rules
- Pokud máš mnoho skillů → vytvoř skill dashboard
- Pokud se skilly překrývají → spusť audit duplicit
- Pokud dashboard používáš opakovaně → automatizuj refresh
- Pokud skill nemá jasné použití → uprav, slouč nebo odstraň
- Katalog zrychluje výběr správného skillu
- Audit snižuje skill sprawl

## Use for
- správa skill knihovny
- review agentického setupu
- kontrola duplicit
- příprava promptu pro workflow

## Avoid
- instalovat skilly bez katalogu
- držet překrývající se skilly
- automatizovat audit před ověřením ručního běhu
